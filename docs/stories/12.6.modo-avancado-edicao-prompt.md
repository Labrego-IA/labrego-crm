# Story 12.6 — Modo Avancado: Edicao Direta do Prompt

**Status:** Ready for Review
**Epic:** 12 — Wizard Gamificado de Configuracao do Agente de Voz
**Priority:** Low
**Complexity:** Low (3 points)
**Depends On:** Story 12.4

---

## Story

Como **usuario tecnico (power user)**, eu quero ter a opcao de ver e editar diretamente o prompt completo gerado pelo wizard, para fazer ajustes finos que o wizard nao cobre.

---

## Problem

O wizard simplifica a configuracao para usuarios nao-tecnicos, mas usuarios avancados podem querer editar o prompt final diretamente — ajustar fraseados especificos, adicionar instrucoes customizadas, ou reescrever secoes inteiras. Sem essa opcao, power users ficam limitados.

---

## Acceptance Criteria

- [x] AC1: Toggle "Modo Avancado" visivel abaixo do wizard (discreto, nao proeminente)
- [x] AC2: DADO que o usuario ativa o modo avancado, QUANDO a tela muda, ENTAO ve o prompt completo em um textarea monospace editavel
- [x] AC3: DADO que o usuario edita o prompt no modo avancado, QUANDO salva, ENTAO o `systemPrompt` salvo e o editado (nao o gerado pelo wizard)
- [x] AC4: DADO que o usuario editou manualmente, QUANDO volta ao wizard, ENTAO ve aviso: "O prompt foi editado manualmente. Gerar novamente pelo wizard vai sobrescrever as edicoes."
- [x] AC5: Botao "Regenerar do Wizard" disponivel no modo avancado para voltar ao prompt gerado
- [x] AC6: TypeScript compila sem erros
- [x] AC7: ESLint passa sem warnings

---

## Scope

### IN Scope
- Toggle para modo avancado
- Textarea monospace para edicao direta
- Logica de prioridade: se editado manualmente, salvar como override
- Aviso ao voltar ao wizard
- Botao regenerar

### OUT of Scope
- Syntax highlighting avancado (VS Code embed)
- Versionamento de prompts
- Diff entre versao wizard e editada

## Dev Notes

**Flag de edicao manual:**
```typescript
// Em AgentWizardAnswers
manuallyEdited?: boolean  // true se o usuario editou o prompt diretamente

// Ao salvar no modo avancado
wizardAnswers.manuallyEdited = true

// Ao regenerar do wizard
wizardAnswers.manuallyEdited = false
```

**Nota:** Esta story pode ser cortada do MVP sem impacto funcional. O wizard sozinho ja atende 90% dos usuarios.
