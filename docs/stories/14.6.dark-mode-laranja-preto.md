# Story 14.6 — Dark Mode com Laranja/Preto

**Status:** Ready for Review
**Epic:** 14 — Redesign Visual: Identidade Labrego (Laranja/Branco/Preto)
**Priority:** Low
**Complexity:** Medium (5 points)
**Depends On:** Story 14.1, Story 14.2, Story 14.3

---

## Story

Como **usuario que prefere dark mode**, eu quero que o tema escuro use a paleta laranja/preto de forma harmoniosa, para que a experiencia noturna seja confortavel e mantenha a identidade da marca.

---

## Problem

O dark mode atual usa cores indigo sobre fundo `#0f172a` (slate-900). Com a troca para laranja, o dark mode precisa ser revisado: laranja sobre fundo escuro tem caracteristicas diferentes de indigo (mais quente, potencialmente mais agressivo para os olhos). Alem disso, a sidebar ja sera escura no modo light (Story 14.2), entao no dark mode ela precisa se diferenciar do conteudo.

## Dev Notes

**Novas CSS variables `.dark`:**
```css
.dark {
  --background: #0A0A0A;
  --foreground: #E5E5E5;
  --color-primary: #FB923C;        /* laranja 400 — mais visivel em fundo escuro */
  --color-primary-600: #F97316;
  --color-primary-500: #FB923C;
  --color-primary-400: #FDBA74;
  --color-secondary: #171717;
  --color-black: #E5E5E5;          /* invertido */
  --color-light: #171717;
  --color-accent: #FDBA74;
  --gradient-header: linear-gradient(90deg, #C2410C 0%, #EA580C 50%, #F97316 100%);
}
```

**Sidebar dark mode:**
A sidebar ja sera `bg-neutral-950` (#0A0A0A) no light mode (Story 14.2). No dark mode:
- Opcao A: Manter igual (sidebar se funde com background)
- Opcao B: Usar `bg-neutral-900` (#171717) para diferenciar com borda `border-neutral-800`
- **Recomendacao:** Opcao B — sidebar levemente mais clara que o background

**Padrao de substituicao de modifiers dark:**

| De (slate-based) | Para (neutral-based) |
|-------------------|---------------------|
| `dark:bg-slate-800` | `dark:bg-neutral-800` |
| `dark:bg-slate-700` | `dark:bg-neutral-700` |
| `dark:bg-slate-900` | `dark:bg-neutral-900` |
| `dark:text-slate-300` | `dark:text-neutral-300` |
| `dark:text-slate-400` | `dark:text-neutral-400` |
| `dark:border-slate-700` | `dark:border-neutral-800` |
