## Story

Como **usuario que gerencia contatos no funil**, eu quero ver as propostas vinculadas ao contato diretamente no painel lateral (que abre ao clicar no card), para que eu nao precise navegar para outra pagina para consultar as propostas.

---

## Problem

Quando o usuario clica num card do funil, abre um painel lateral com informacoes do contato e anotacoes. Porem, para ver as propostas do contato, ele precisa sair do funil e navegar para `/contatos/{id}` > aba "Propostas". Isso quebra o fluxo de trabalho no kanban.

**Query de propostas:**
```typescript
// Ao abrir painel (selectedClient muda)
useEffect(() => {
  if (!selectedClient) return
  const q = query(
    collection(db, 'proposals'),
    where('clientId', '==', selectedClient.id),
    orderBy('createdAt', 'desc')
  )
  const unsub = onSnapshot(q, snap => {
    setClientProposals(snap.docs.map(d => ({ id: d.id, ...d.data() })))
  })
  return unsub
}, [selectedClient?.id])
```
