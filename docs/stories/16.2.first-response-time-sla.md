## Story

Como **gestor comercial**, eu quero ver o tempo de primeiro contato (First Response Time) de cada vendedor e indicadores de SLA, para que eu possa identificar gargalos no atendimento e garantir respostas rapidas.

---

## Problem

Nao existe metrica de velocidade de atendimento. O gestor nao sabe se leads estao sendo contactados rapidamente. Leads que esperam demais tendem a esfriar. A falta de SLA visual dificulta cobranca e melhoria de processo.

## Dev Notes

**Calculo FRT:**
```typescript
const frt = firstContactAt && createdAt
  ? (new Date(firstContactAt).getTime() - new Date(createdAt).getTime()) / (1000 * 60 * 60)
  : null
```

**Dados existentes:** `firstContactAt` ja existe no tipo Cliente, so nao e usado para metricas.

**Agrupamento por vendedor:** Usar campo `assignedTo`/`assignedToName` para agrupar.
