# Story 12.2 — Wizard Gamificado: Fases 1-3 (Identidade, Negocio, Abertura)

**Status:** Ready for Review
**Epic:** 12 — Wizard Gamificado de Configuracao do Agente de Voz
**Priority:** High
**Complexity:** High (8 points)
**Depends On:** Story 12.1

## Story

Como **usuario que configura o agente de voz**, eu quero responder perguntas simples em linguagem natural sobre meu agente, meu negocio e minha estrategia de abertura, para que o sistema crie automaticamente um prompt profissional sem que eu precise entender de IA.

---

## Problem

A aba "Conhecimento" atual apresenta campos tecnicos sem contexto ("System Prompt", "First Message"). Usuarios nao-tecnicos nao sabem o que preencher e acabam com agentes de voz com prompts fracos. Precisamos de uma experiencia guiada que faca as perguntas certas em linguagem natural.

## Dev Notes

**Perguntas por fase (UX conversacional):**

**Fase 1 — "Vamos dar vida ao seu agente!"**
1. "Qual nome o seu agente vai ter?" — placeholder: "Ex: Leo, Carol, Rafael"
2. "Qual sera o cargo dele na empresa?" — placeholder: "Ex: SDR Senior, Consultor de Vendas"
3. "Qual o nome da sua empresa?" — placeholder: "Ex: Labrego IA"
4. "Como voce quer que ele fale? Descreva o tom de voz" — placeholder: "Ex: Confiante e direto, com calor humano. Fala como alguem que ja viu centenas de operacoes..."

**Fase 2 — "Me conta sobre seu negocio"**
1. "O que sua empresa vende ou oferece?" — textarea, placeholder: "Ex: Automacao de processos com IA para empresas medias..."
2. "Quem e seu cliente ideal? Descreva o perfil" — textarea, placeholder: "Ex: Empresas de 10 a 200 funcionarios com processos manuais..."
3. "O que diferencia voces da concorrencia?" — textarea
4. "Qual o principal beneficio para quem contrata voces?" — textarea

**Fase 3 — "A primeira impressao"**
1. "Como o agente deve se apresentar nos primeiros 30 segundos?" — textarea grande, placeholder: "Ex: Ola {{contactName}}, aqui e o Leo da Labrego IA. Me da 30 segundos pra explicar..."
2. "Qual gancho usar para prender a atencao do prospect?" — textarea, placeholder: "Ex: A gente ajuda empresas como a sua a automatizar processos que consomem tempo..."

**Auto-save:**
```typescript
// Cada campo faz save no onBlur
const handleFieldBlur = async (field: string, value: string) => {
  const updated = { ...wizardAnswers, [field]: value, lastUpdated: new Date().toISOString() }
  // Salvar apenas o wizardAnswers no Firestore (merge)
  await updateDoc(configRef, { 'agentKnowledge.wizardAnswers': updated })
}
```
