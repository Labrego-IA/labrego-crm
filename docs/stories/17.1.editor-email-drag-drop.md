# Story 17.1 — Editor de Email Drag-and-Drop

**Status:** Ready for Review
**Epic:** 17 — Email Marketing: Editor & Templates
**Priority:** Medium
**Complexity:** High (13 points)
**Depends On:** Story 4.1

---

## Story

Como **gestor de marketing**, eu quero um editor visual de email com drag-and-drop para compor campanhas bonitas sem saber HTML, para que eu possa criar comunicacoes profissionais rapidamente.

---

## Problem

O editor de email atual e um textarea simples. Emails enviados parecem plain text, sem formatacao profissional. O gestor precisa pedir ajuda de designer/dev para criar emails bonitos. Concorrentes como RD Station e Mailchimp oferecem editores visuais.

---

## Risks

- Complexidade alta (13pts) — maior story do backlog, pode precisar ser quebrada
- Compatibilidade de HTML entre email clients e notoriamente problematica (Outlook, Gmail, Apple Mail)
- Biblioteca externa (Unlayer) pode ter limitacoes de customizacao ou licenca
- Performance do editor com muitos blocos pode degradar em mobile
- Inline CSS conversion (juice) pode nao cobrir todos os casos de email clients

---

## Dev Notes

**Abordagem recomendada:** Usar biblioteca `react-email-editor` (Unlayer) ou construir editor custom simples com DnD Kit.

**HTML para email:** Emails precisam de inline styles (nao suportam CSS externo). Usar `juice` para converter.

**Variaveis:** Substituir `{{nome}}` por campo do contato antes do envio.

**Collection:** Salvar templates em `emailTemplates` (top-level, orgId).
