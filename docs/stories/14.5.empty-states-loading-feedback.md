**Status:** Ready for Review

## Story

Como **usuario do CRM**, eu quero ver feedback visual claro quando nao ha dados, quando algo esta carregando, ou quando algo deu errado, para que eu nunca fique olhando para uma tela vazia sem saber o que fazer.

---

## Problem

Hoje o CRM tem:
1. **Classe `.empty-state` em CSS** mas nenhum componente React reutilizavel — cada pagina improvisa
2. **Classe `.skeleton` em CSS** com animacao shimmer mas nenhum componente pronto
3. **Loading splash** (`Loading.tsx`) com logo + spinner — funciona, mas nao ha skeleton loaders para loading parcial
4. **Toast** via Sonner funciona bem, mas mensagens de erro sao genericas
5. Paginas que falham silenciosamente (ex: analytics sem contatos mostra tela vazia)

## Dev Notes

**Componente EmptyState:**
```tsx
interface EmptyStateProps {
  icon?: React.ReactNode
  title: string
  description?: string
  action?: {
    label: string
    onClick: () => void
  }
}

export function EmptyState({ icon, title, description, action }: EmptyStateProps) {
  return (
    <div className="empty-state">
      {icon && <div className="text-neutral-300">{icon}</div>}
      <h3 className="text-base font-semibold text-neutral-700">{title}</h3>
      {description && <p className="text-sm text-neutral-400 max-w-sm">{description}</p>}
      {action && (
        <button onClick={action.onClick} className="btn-primary mt-2">
          {action.label}
        </button>
      )}
    </div>
  )
}
```

**Componente Skeleton:**
```tsx
interface SkeletonProps {
  variant?: 'text' | 'card' | 'avatar' | 'table-row'
  className?: string
  count?: number  // repetir N vezes
}

// Variantes:
// text:       h-4 w-full rounded
// card:       h-32 w-full rounded-2xl
// avatar:     h-10 w-10 rounded-full
// table-row:  h-8 w-full rounded (grid de 4 colunas)
```

**Exemplos de uso:**
```tsx
// Contatos loading
{loading ? (
  <div className="space-y-3">
    <Skeleton variant="table-row" count={5} />
  </div>
) : contacts.length === 0 ? (
  <EmptyState
    icon={<UserGroupIcon className="w-16 h-16" />}
    title="Nenhum contato encontrado"
    description="Comece adicionando seu primeiro contato ao CRM."
    action={{ label: 'Criar contato', onClick: () => setShowModal(true) }}
  />
) : (
  <ContactsTable contacts={contacts} />
)}
```

**Classe `.skeleton` ja existe em globals.css** (linhas 220-242) com animacao shimmer. O componente React apenas encapsula essas classes.
