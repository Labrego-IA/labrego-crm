# Story 14.3 — Componentes Core: Botoes, Cards, Inputs, Badges, Tabs

**Status:** Ready for Review
**Epic:** 14 — Redesign Visual: Identidade Labrego (Laranja/Branco/Preto)
**Priority:** High
**Complexity:** High (8 points)
**Depends On:** Story 14.1

## Story

Como **usuario do CRM**, eu quero que botoes, cards, inputs e badges sigam a nova identidade visual laranja/branco/preto, para que toda interacao com o sistema seja consistente e profissional.

---

## Problem

Os componentes core (botoes, cards, inputs, badges, tabs) usam a cor indigo via tokens (`primary`). Apos a Story 14.1 trocar a paleta, eles ja ficarao laranja automaticamente, mas precisam de ajustes finos de contraste, sombras e espacamento para ficar premium com a nova paleta. Alem disso, o componente Tabs usa gradiente hardcoded (indigo → violet → sky) que precisa ser atualizado.

## Dev Notes

**Contraste WCAG AA:**
- `#F97316` (laranja 500) sobre branco: ratio ~2.8:1 — **FALHA para texto pequeno**
- `#EA580C` (laranja 600) sobre branco: ratio ~4.0:1 — **Proximo**
- `#C2410C` (laranja 700) sobre branco: ratio ~5.7:1 — **PASSA**
- Branco sobre `#F97316`: ratio ~2.8:1 — **FALHA para texto pequeno**
- Branco sobre `#EA580C`: ratio ~4.0:1 — **Proximo**

**Decisao de contraste:** Para botoes, o texto branco sobre `#EA580C` (600) e aceitavel porque botoes usam `font-semibold` (texto grande = ratio 3:1 minimo). Para texto corrido, usar `#C2410C` (700) ou mais escuro.

**Tabs.tsx — gradiente atual (a substituir):**
```tsx
// ANTES (hardcoded indigo):
className="bg-gradient-to-r from-indigo-500 via-violet-500 to-sky-400"

// DEPOIS (laranja quente):
className="bg-gradient-to-r from-orange-600 via-orange-500 to-amber-400"
// Ou usar primary tokens:
className="bg-primary"
```

**ui.ts — ja usa tokens `primary`:**
Os estilos em `ui.ts` ja referenciam `primary` (ex: `bg-primary/10`, `text-primary`), entao a troca do tailwind.config (Story 14.1) ja resolve a maioria. Verificar apenas se ha alguma classe hardcoded.

**Componentes a verificar:**
- `src/components/Tabs.tsx` — gradiente hardcoded
- `src/components/Modal.tsx` — focus ring, botoes
- `src/components/FilterBar.tsx` — possiveis cores hardcoded
- `src/components/MemberSelector.tsx` — possiveis cores hardcoded
- `src/components/shared/PaginationControls.tsx` — active state
