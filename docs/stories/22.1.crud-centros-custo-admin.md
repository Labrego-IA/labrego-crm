# Story 22.1 — CRUD Centros de Custo (Admin)

**Status:** Ready
**Epic:** 22 — Centros de Custo
**Points:** 3
**Priority:** Medium
**Dependencies:** Nenhuma

---

## Story

Como administrador do CRM, quero ter uma página no painel de administração para criar, editar e excluir centros de custo, para categorizar os contatos e negócios por área de responsabilidade financeira.

---

## Acceptance Criteria

### AC1: Nova página de admin para centros de custo
- **Given** o usuário com permissão de admin
- **When** acessa `/admin/centros-custo`
- **Then** vê a lista de centros de custo existentes da organização
- **And** a página está disponível no sidebar de administração

### AC2: Criar centro de custo
- **Given** a página de centros de custo
- **When** o usuário clica em "Novo Centro de Custo"
- **Then** abre um modal/formulário com campos:
  - **Nome** (obrigatório, texto)
  - **Descrição** (opcional, texto)
  - **Cor** (opcional, seletor de cor — palette pré-definida)
  - **Ativo** (toggle, default true)
- **And** ao salvar, cria documento em `organizations/{orgId}/costCenters`
- **And** exibe toast de sucesso

### AC3: Editar centro de custo
- **Given** um centro de custo na lista
- **When** o usuário clica no botão de editar
- **Then** abre o mesmo formulário preenchido com os dados atuais
- **And** ao salvar, atualiza o documento
- **And** exibe toast de sucesso

### AC4: Excluir centro de custo
- **Given** um centro de custo na lista
- **When** o usuário clica em excluir
- **Then** exibe confirmação "Tem certeza que deseja excluir este centro de custo?"
- **And** ao confirmar, deleta o documento
- **And** exibe toast de sucesso
- **And** NÃO remove costCenterId dos clientes (orphan cleanup não é necessário)

### AC5: Listagem com informações
- **Given** centros de custo cadastrados
- **When** a lista é renderizada
- **Then** mostra para cada item:
  - Cor (badge/dot)
  - Nome
  - Descrição
  - Status (Ativo/Inativo)
  - Contagem de contatos associados (query count)
  - Botões de ação (editar, excluir)

---

## Scope

### IN Scope
- Rota `/admin/centros-custo`
- CRUD completo (Create, Read, Update, Delete)
- Item no sidebar de admin
- Collection: `organizations/{orgId}/costCenters`
- Seletor de cor pré-definido (palette)
- Contagem de contatos por centro de custo

### OUT of Scope
- Relatórios financeiros por centro de custo
- Atribuição em massa de centros de custo
- Hierarquia de centros de custo
