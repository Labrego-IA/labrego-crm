## Story

Como **desenvolvedor que vai construir o wizard gamificado**, eu quero ter os tipos TypeScript, a funcao de montagem de prompt e a formula de forca do agente definidos e testados, para que as stories de UI possam consumir essa fundacao.

## Problem

Hoje o `systemPrompt` e um textarea cru que o usuario preenche manualmente. Nao existe logica para montar um prompt profissional estruturado a partir de respostas em linguagem natural. Sem essa fundacao, as stories de UI do wizard nao tem o que consumir.

## Dev Notes

**Tipo AgentWizardAnswers (estrutura):**
```typescript
interface AgentWizardAnswers {
  // Fase 1: Identidade
  agentName: string
  agentRole: string
  companyName: string
  toneDescription: string  // "Como voce quer que o agente fale?"

  // Fase 2: Negocio
  whatYouSell: string       // "O que sua empresa vende/oferece?"
  idealCustomer: string     // "Descreva seu cliente ideal"
  differentials: string     // "O que diferencia voces?"
  valueProposition: string  // "Qual o principal beneficio?"

  // Fase 3: Abertura
  openingApproach: string   // "Como o agente deve se apresentar?"
  hookStrategy: string      // "Qual gancho usar para prender atencao?"

  // Fase 4: Investigacao
  discoveryQuestions: string[] // "Perguntas para entender a dor"
  qualificationCriteria: string // "Criterios para qualificar"

  // Fase 5: Proposta & Agendamento
  solutionBridge: string    // "Como conectar dor a solucao?"
  specialistName: string    // "Nome do especialista para reuniao"
  meetingDuration: number   // minutos

  // Fase 6: Objecoes
  objections: { objection: string; response: string }[]

  // Fase 7: Linguagem & Regras
  forbiddenWords: string    // Texto livre, separado por virgula
  keyExpressions: string    // Texto livre, separado por virgula
  behaviorRules: string     // Regras extras em linguagem natural

  // Meta
  completedPhases: number[] // [1,2,3...]
  strengthScore: number     // 0-100, calculado
  lastUpdated: string       // ISO date
}
```

**Referencia de prompt alvo:** O prompt LEO fornecido pelo usuario (~4.500 palavras) serve como template de qualidade. A funcao `assemblePromptFromWizard()` deve gerar prompts com a mesma estrutura e profundidade.

**Formula Agent Strength:**
```
strength = sum(peso[fase] * completude[fase]) / sum(pesos) * 100

Pesos: { 1: 20, 2: 20, 3: 10, 4: 15, 5: 15, 6: 15, 7: 5 }

completude[fase] = campos_preenchidos_na_fase / total_campos_na_fase
```
