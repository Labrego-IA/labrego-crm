# Story 21.1 — Deal Value no Card + KPIs de Valor no Funil

## Story

Como usuário do CRM, quero visualizar o valor do negócio diretamente no card do Kanban e ter KPIs financeiros tanto no header do funil quanto em cada coluna de etapa, para ter visibilidade imediata do pipeline de vendas sem precisar abrir o detalhe do cliente.

---

## Acceptance Criteria

### AC1: Deal Value abreviado no KanbanCard
- **Given** um cliente com `dealValue > 0`
- **When** o card é renderizado no Kanban
- **Then** exibe um badge com ícone `$` e valor abreviado:
  - `< 1000` → ex: `$500`
  - `>= 1000 e < 1M` → ex: `$12K`, `$150K`
  - `>= 1M` → ex: `$1.2M`
- **And** se `dealValue` é null/0/undefined, o badge NÃO aparece
- **And** o badge usa cor verde (emerald-600) para destacar valor

### AC2: KPIs financeiros no header do funil
- **Given** o funil `/funil/[funnelId]` carregado com clientes
- **When** o header (globalMetrics) é renderizado
- **Then** exibe dois novos KPIs:
  1. **Valor Total Pipeline**: soma de `dealValue` de TODOS os clientes em etapas que `countsForMetrics === true`
  2. **Valor Esperado**: soma de `(dealValue × probability / 100)` para cada cliente, onde `probability` vem da etapa do cliente (campo `stage.probability`)
- **And** valores formatados em BRL abreviado (ex: R$ 125K, R$ 1.2M)
- **And** se valor = 0, exibe "R$ 0"

### AC3: KPIs financeiros em cada coluna de etapa
- **Given** uma coluna de etapa no Kanban
- **When** renderizada com clientes
- **Then** o header da coluna exibe:
  1. **Valor total**: soma dos `dealValue` dos clientes naquela etapa
  2. **Valor esperado**: soma dos `(dealValue × stage.probability / 100)` dos clientes naquela etapa
- **And** exibe abaixo do count de contatos, em fonte menor
- **And** formatação abreviada em BRL

### Função de formatação

```typescript
function formatCurrencyShort(value: number): string {
  if (!value) return ''
  if (value >= 1_000_000) return `R$ ${(value / 1_000_000).toFixed(1)}M`
  if (value >= 1_000) return `R$ ${Math.round(value / 1_000)}K`
  return `R$ ${Math.round(value)}`
}
```
