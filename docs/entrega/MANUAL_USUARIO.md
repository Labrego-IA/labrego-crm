# Manual do Usuario — Labrego CRM

**Versao:** 1.0.0

---

## Acesso

1. Acesse a URL do sistema
2. Faca login com email e senha

---

## Pipeline de Vendas

1. Menu → **Pipeline**
2. Visualize negocios em colunas (Kanban)
3. Arraste negocios entre etapas
4. Clique para ver detalhes

---

## Contatos

1. Menu → **Contatos**
2. Cadastre e gerencie contatos
3. Associe a empresas e negocios
4. Registre interacoes

---

## Emails

1. Abra um contato ou negocio
2. Use o editor rico para compor emails
3. Aplique templates predefinidos
4. Envie e acompanhe historico

---

## Chamadas

- Registre ligacoes recebidas e realizadas
- Associe a contatos e negocios
- Acompanhe metricas de chamadas

---

## Relatorios

- Gere relatorios PDF de performance
- Exporte dados do pipeline

---

## Chrome Extension

- Instale a extensao no navegador
- Capture dados de paginas web diretamente para o CRM
