# Guia de Instalação - Extensão Gestão Mundo para WhatsApp

Este guia explica como instalar a extensão do Chrome que integra o WhatsApp Web com o CRM Gestão Mundo.

---

## Requisitos

- Google Chrome (versão 88 ou superior)
- Conta ativa no Gestão Mundo

---

## Passo 1: Baixar a Extensão

1. Baixe o arquivo `gestao-mundo-extension.zip`
2. Extraia o conteúdo em uma pasta no seu computador
   - **Windows**: Clique com botão direito > "Extrair tudo"
   - **Mac**: Clique duplo no arquivo .zip

> **Importante**: Lembre-se onde salvou a pasta extraída, você vai precisar dela no próximo passo.

---

## Passo 2: Ativar o Modo Desenvolvedor no Chrome

1. Abra o Google Chrome
2. Digite na barra de endereços: `chrome://extensions/`
3. Pressione **Enter**
4. No canto superior direito, ative o **Modo do desenvolvedor**

![Ativar modo desenvolvedor](https://i.imgur.com/example.png)

---

## Passo 3: Instalar a Extensão

1. Na mesma página (`chrome://extensions/`), clique no botão **"Carregar sem compactação"**
2. Navegue até a pasta onde você extraiu os arquivos
3. Selecione a pasta `chrome-extension` (ou a pasta que contém o arquivo `manifest.json`)
4. Clique em **"Selecionar pasta"**

A extensão será instalada e você verá o ícone dela na barra de extensões do Chrome.

---

## Passo 4: Configurar a Extensão

1. Clique no ícone da extensão na barra do Chrome (pode estar no menu de extensões 🧩)
2. Se necessário, fixe a extensão clicando no pin 📌
3. No popup que abrir, verifique se a URL do CRM está correta:
   - URL padrão: `https://www.mundofeat.app`
4. Clique em **"Fazer Login"**
5. Entre com seu **email** e **senha** do Gestão Mundo

---

## Passo 5: Usar a Extensão

1. Acesse o WhatsApp Web: https://web.whatsapp.com
2. Após conectar seu WhatsApp, você verá um **botão azul flutuante** no canto inferior direito
3. Clique nele para abrir o painel da extensão

### Atalho de Teclado
- **Windows/Linux**: `Ctrl + Shift + G`
- **Mac**: `Cmd + Shift + G`

---

## O Que Você Pode Fazer

### Criar Contatos
1. Abra uma conversa no WhatsApp
2. Clique no botão flutuante
3. Os dados do contato são preenchidos automaticamente
4. Complete as informações e clique em **"Criar Contato"**

### Salvar Mensagens no CRM
1. No painel, vá para a aba **"Mensagens"**
2. Clique em **"Ativar Seleção"**
3. Clique nas mensagens que deseja salvar (elas ficam destacadas)
4. Escolha a oportunidade para vincular
5. Clique em **"Salvar no CRM"**

### Copiar Mensagens
1. Selecione as mensagens desejadas
2. Clique em **"Copiar Selecionadas"**
3. As mensagens são copiadas para a área de transferência

---

## Solução de Problemas

### A extensão não aparece no WhatsApp Web
- Recarregue a página do WhatsApp Web (F5)
- Verifique se a extensão está ativa em `chrome://extensions/`

### Erro ao fazer login
- Verifique se seu email e senha estão corretos
- Certifique-se de que você tem uma conta ativa no Gestão Mundo

### Botão flutuante sumiu
- Use o atalho `Ctrl + Shift + G` (ou `Cmd + Shift + G` no Mac)
- Recarregue a página do WhatsApp Web

### Não consigo salvar mensagens
- Verifique se você está logado na extensão
- Certifique-se de ter selecionado uma oportunidade

---

## Atualizando a Extensão

Quando houver uma nova versão:

1. Baixe o novo arquivo .zip
2. Extraia na mesma pasta (substituindo os arquivos antigos)
3. Vá para `chrome://extensions/`
4. Clique no botão de **atualizar** (🔄) na extensão Gestão Mundo
5. Recarregue o WhatsApp Web
